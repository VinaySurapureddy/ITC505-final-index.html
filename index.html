Lights Out

Turn off all the lights to win the game!

Game Rules

  • The grid has lights that are either on (dark) or off (white).
  • Clicking a cell toggles that cell and its adjacent neighbors.
  • Your goal is to turn all lights off.
  • The puzzle is randomly generated but always solvable.

Development Addendum

Process: Designed a 5x5 grid using HTML/CSS. Each cell toggles its state and neighbors via JavaScript.

Solvability: Board starts solved; simulated random clicks ensure each configuration is winnable.

Challenges: Win-check logic was tricky, solved by checking if all cells are off after each move.

Learnings: Improved JS event handling, DOM manipulation, and use of CSS Grid for layout.
